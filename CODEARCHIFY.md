#### üìñ Context

* This repository contains a Lambda function extension that captures Lambda invocation and performance telemetry and pushes it to an Amazon Kinesis data stream.
* Services, SDKs, APIs, databases, cloud providers, and third-party tools used:
  * AWS Lambda
  * AWS Kinesis
  * AWS Systems Manager Parameter Store
  * AWS Secrets Manager (implied)
  * AWS CloudFormation
  * Node.js

#### üìñ Overview

* The Lambda function extension captures invocation and performance telemetry and pushes it to a Kinesis data stream.
* The extension registers with the Lambda runtime API, receives telemetry events, and sends them to the Kinesis data stream.

#### üîπ Components

| --------------------------- | ------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------- |
| Lambda Function Extension   | A Lambda extension that captures Lambda invocation and performance telemetry                | Lambda Runtime API, Kinesis Data Stream                                                         | Registers with the Lambda runtime API, receives telemetry events, and sends them to the Kinesis data stream. |

| Lambda Runtime API         | An API that enables Lambda extensions to interact with the Lambda runtime environment      | Lambda Function Extension                                                                       | Allows the Lambda Function Extension to register and receive telemetry events.                          |
| Systems Manager Parameter Store | A service that enables you to store, manage, and retrieve parameter values                 | Lambda Function Extension                                                                       | Stores the ARN of the Kinesis data stream and the managed policy ARN for the Lambda Function Extension. |
